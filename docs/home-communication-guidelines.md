# Home Communication Guidelines

Date: 2026-03-08
Project: Mis Incentivos 2.0 Novartis

## Goal
Position the home/login page as an empowerment and clarity touchpoint for the sales force, not only as an access form.

## Audience
- Primary: Novartis sales force.
- Secondary: Corporate/admin users (not the main voice for home copy).

## Voice Attributes
- Empowering
- Clear and direct
- Innovative
- Inspiring

## Core Value Proposition
Mis Incentivos 2.0 transforms effort into clear results and recognizes individual impact in Novartis mission.

## Home Structure
- Split layout: branding/marketing left, login right.
- Mobile first: single column stack on small screens.

## Recommended Copy
- Headline: "Tu impacto, mas claro que nunca."
- Alternate headline: "Bienvenido a la nueva era de tus incentivos."
- Subheadline: "Mas rapida, segura e intuitiva. Disenada para ti."
- Purpose line: "Tu exito impulsa nuestra mision. Juntos, reimaginamos la medicina."
- Login title: "Bienvenido"
- Login subtitle: "Inicia sesion para continuar"
- Primary CTA: "Ingresar"
- SSO CTA: "Ingresar con tu cuenta Novartis (SSO)"

## Visual Direction
- Main palette:
  - Midnight Blue: #002068
  - Cinnabar: #E74A22
  - Carrot Orange: #ED9A1E
  - Near Black: #161616
  - Warm White: #FCFCFC
- Use generous whitespace and simple, high-contrast layout.
- Use line-art style iconography for benefits if needed.

## UX Priorities
- Fast login path with minimal friction.
- Transparent message hierarchy and short copy.
- Full-width composition with controlled content containers.
- Authentication model: Supabase magic link only.
- Allowed login domains: `@novartis.com` and `@jelpus.com`.
